# traefik-test
A small study project on [traefik](https://github.com/containous/traefik), a reverse proxy server.

## Calling whoami

```
$ curl -H Host:whoami.docker.localhost http://127.0.0.1
$ http http://localhost:80 Host:whoami.docker.localhost
```
